# DDCGPT-OS

A browser-based “ChatGPT OS” concept for creator workflows — GPT as the operating layer for file actions, organization, and on-the-fly content/photo workflows.

## What it is
DDCGPT-OS is an early prototype exploring a desktop-like UI powered by ChatGPT:
- **GPT File Manager:** natural language actions (organize, rename, summarize, tag)
- **GPT Photo Gallery:** edit/augment metadata + generate edits/workflows on the fly
- **OS-style workspace:** “apps” and workspaces instead of a single chat window

## Status
🚧 Early prototype / active development. Not production-ready.

## Why
I wanted a creator-first interface where ChatGPT isn’t just a chat box — it’s the operating layer.
